# Edge Detection

A C++/OpenCV image-processing project exploring classical edge-detection and filtering techniques.

This project was built as part of my computer vision and image-processing learning/research work. It demonstrates practical implementation and experimentation with low-level image-processing workflows rather than relying only on black-box library calls.

## What it explores

The project includes experiments with:

- Sobel edge detection
- Canny edge detection
- Laplacian filtering
- Gaussian filtering
- gradient direction analysis
- image noise experiments
- grayscale image processing
- webcam/video-frame processing experiments

## Technical focus

This repository demonstrates:

- C++ programming
- OpenCV usage
- image-processing fundamentals
- convolution kernels
- edge and gradient analysis
- experimentation with multiple detection approaches
- practical debugging and visual inspection of image outputs
